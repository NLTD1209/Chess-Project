# Chess Project
For USTH Software Engineering course 2020 - Group 5

## Installation
This game require Python 3 and the *pygame* library.
Get *pygame* using this command:
```
pip install pygame
```
## Startup
Use the command:
```
python gui.py
```
or
```
python3 gui.py
```
## Application feature
The software is a simple chess game written in Python.
